# Distributed Bucket Sort — MPI & OpenMP

## Tagline
High-performance distributed sorting with MPI, benchmarked across dataset sizes and process counts.

## Description
A distributed implementation of Bucket Sort and Top-K extraction using MPI in C, with a hybrid MPI+OpenMP variant for multi-level parallelism. Includes comprehensive benchmark scripts, CSV result exports, and Python visualization of speedup curves across varying dataset sizes and process counts.

## Motivation
Master 1 parallel programming project demonstrating MPI communication patterns, distributed algorithms, and hybrid parallelism strategies for high-performance computing.

## Tech Stack
- **Languages**: C, Python 3
- **Parallel**: MPI (OpenMPI), OpenMP
- **Visualization**: Matplotlib, Pandas, NumPy
- **Build**: Makefile, Bash scripts

## Key Features
- Distributed Bucket Sort with MPI (broadcast, scatter, gather)
- Top-K extraction variant with early termination optimization
- Hybrid MPI+OpenMP version for multi-level parallelism
- Comprehensive benchmarking with CSV result export
- Python-generated speedup curve visualizations
